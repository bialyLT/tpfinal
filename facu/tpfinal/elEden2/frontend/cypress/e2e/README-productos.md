# Productos (productos.cy.js)

## Objetivo
- Validar que un empleado/admin puede acceder al inventario.

## Precondiciones
- Usuario con rol empleado o admin.
- Backend disponible.
- BaseUrl de Cypress apuntando al frontend.

## Pasos automatizados
- Login desde UI.
- Navegar por sidebar a Inventario > Productos.
- Validar pagina de Inventario de Productos.

## Datos
- Usuario existente con rol.

## Variables
- E2E_EMAIL / E2E_PASSWORD
- BaseUrl de Cypress configurada.
